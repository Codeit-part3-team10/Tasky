### 💁🏻‍♀️💁🏻‍♂️ 프로젝트 소개
- Taskify는 다른 사용자들과 함께 일정을 관리하고 공유할 수 있는 서비스입니다.
- 모든 서비스는 회원가입을 한 후에 로그인을 해야만 사용할 수 있습니다.
- 사용자들은 대시보드를 생성할 수 있고 각 대시보드에서 일정을 생성할 수 있으며 대시보드에 원하는 사용자를 초대할 수 있습니다.
